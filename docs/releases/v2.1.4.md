# Legal Extractor v2.1.4 更新日志

## ✨ 新特性 (New Features)
- **专业版激活流程优化**：
  - 新增激活状态的实时反馈与动画效果
  - 优化了 macOS 下系统权限请求的交互时机
  - 增加了对授权码格式的基础校验

## 🐛 问题修复 (Bug Fixes)
- 修复了在首次安装且无配置文件时，激活失败并报错 `Config File "conf" Not Found` 的问题
- 修复了提示框 (Toast) 被激活弹窗遮挡的问题
- 修复了“正在处理中”文案在激活时错误显示为“正在解析文档”的问题
- 修复了多次点击激活按钮导致提示框闪烁的问题
- 修复了弹窗关闭后输入框未自动清空的问题

## 📦 下载与安装
- macOS (Intel/M1/M2): 下载 `.dmg` 文件
- Windows: 下载 `.exe` 文件
