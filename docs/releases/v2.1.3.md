# LegalExtractor V2.1.3

本次更新重点优化了前端 UI/UX，提升了无障碍体验和视觉交互的稳定性。

### ✨ UI/UX 优化
- **无障碍支持**: 为所有输入控件添加了 `aria-label`，为错误提示添加了 `role="alert"`，并支持 `prefers-reduced-motion` 媒体查询。
- **图标系统升级**: 将界面中残留的 Emoji 图标全部替换为矢量 SVG 图标，视觉更加专业统一。
- **交互性能提升**: 优化了 hover 动画效果，解决了部分组件缩放导致的布局抖动问题。
- **视觉增强**: 添加了更平滑的加载遮罩层样式和状态闪烁动画。

### 📦 升级说明
建议所有用户升级以获得更流畅的交互体验。
