# LegalExtractor V2.1.0

本次更新标志着 LegalExtractor 从单一桌面端工具正式演进为 **Web/Desktop 双模架构** 系统。

### 🚀 新增特性
- **Web 模式支持**: 新增基于 Docker 的 Web 版，支持在浏览器中直接上传、预览和提取法律文书。
- **REST API 接口**: 提供标准 HTTP 接口 (`/api/extract`, `/api/export`)，方便集成到现有工作流。
- **环境自适应前端**: 自动识别 Wails 桌面环境与浏览器环境，无缝切换 API 调用方式。
- **IO 解耦重构**: 核心提取引擎全面支持内存数据流，彻底摆脱对本地文件系统的依赖。

### 🛠️ 优化与修复
- **配置热加载**: 引入 `Air` 工具支持后端开发热重载。
- **智能配置回退**: 支持自动加载内置 `baked_conf.yaml`，简化初次部署成本。
- **UI 细节优化**: 修复了 Web 模式下文件对象显示为 `[object File]` 的问题。
- **项目瘦身**: 移除了旧版遗留的 Python 依赖及构建脚本。

### 🐳 Web 版快速启动
```bash
docker-compose up -d
```
详情请参阅 `README.md`。
