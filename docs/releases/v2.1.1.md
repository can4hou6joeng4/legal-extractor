# LegalExtractor V2.1.1

本次更新修复了百度 AI 文档解析的关键问题，并完善了 CI/CD 密钥管理。

### 🐛 Bug 修复
- **修复百度 API 解析失败**: 修正 `file_name` 参数传递完整路径导致的 "图片为空" 错误，现在只传递文件名。
- **完善错误信息捕获**: 添加 `task_error` 字段解析，API 失败时显示详细错误原因。
- **添加文件有效性检查**: 在提交到百度 API 前检查文件内容是否为空。

### 🔐 CI/CD 改进
- **GitHub Secrets 集成**: 构建时自动从 Secrets 注入百度 API 密钥到产物中。
- **用户零配置体验**: 下载的 DMG/EXE 开箱即用，无需额外配置 API 密钥。

### 📦 升级说明
直接下载新版本即可，无需任何配置更改。
