# Legal Extractor v2.1.5 更新日志

## ✨ 新特性 (New Features)
- **OCR 服务升级**：
  - 将 OCR 服务从百度云迁移至腾讯云 SmartStructuralOCRV2
  - 采用同步返回模式，响应速度更快
  - 使用固定字段提取，识别精度更高

## 🔧 优化改进 (Improvements)
- **配置简化**：
  - 配置项从 3 个减少到 2 个（仅需 SecretId 和 SecretKey）
  - 移除异步轮询逻辑，代码更简洁
- **数据清理**：
  - 优化被告字段提取，自动移除末尾多余标点符号
  - 身份证号自动过滤非法字符

## 📖 文档更新 (Documentation)
- 配置指南已更新为腾讯云版本
- 新增腾讯云 API 密钥申请步骤说明

## 📦 下载与安装
- macOS (Intel/M1/M2): 下载 `.dmg` 文件
- Windows: 下载 `.exe` 文件

## 🔑 配置说明
使用前请配置腾讯云 API 密钥，详见 [配置指南](../user/CONFIG_GUIDE.md)
