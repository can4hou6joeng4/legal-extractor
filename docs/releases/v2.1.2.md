# LegalExtractor V2.1.2

本次更新新增软件试用期功能，增强了错误处理机制，并完善了测试与文档。

### ✨ 新功能
- **7 天试用期倒计时**: 桌面版顶部显示试用剩余时间，试用期结束后核心提取功能将被锁定。
- **智能错误提示**: 针对常见 API 错误（网络超时、配额用尽、文件加密等）提供友好的中文提示和解决建议。
- **自动重试机制**: 网络波动或 Token 过期时自动重试，提升稳定性。

### 🐛 Bug 修复
- **Windows 构建修复**: 修复 Windows 平台 CI/CD 构建缺少 BuildTime 注入的问题，确保试用期功能正常工作。

### 📚 文档与测试
- **故障排查指南**: 新增 `TROUBLESHOOTING.md`，涵盖 PDF 解析、文件上传、导出等常见问题的解决方案。
- **单元测试覆盖**: 添加 17 个测试用例，覆盖文件名处理、API 响应解析等核心逻辑。

### 📦 升级说明
直接下载新版本即可。试用期从构建时间开始计算，共 7 天。
